# 游戏设计用户研究工具 v2.0

> 🎮 一款面向游戏设计专业学生的用户研究辅助工具

## 作者

**雷杰茗**

## 功能特性

| 模块 | 功能 |
|------|------|
| 竞品分析 | 游戏对比评分 + 文字描述 + 雷达图可视化 + 6 大品类模板 |
| 用户画像 | 问卷式访谈 + 玩家类型测试（巴特尔模型）+ 问题顺序拖拽调整 |
| 用户访谈 | 提纲模板 + 访谈记录 + 洞察提炼 |
| 问卷设计 | 6 种模板 + 4 种题目类型 + 5 种维度分类 + 拖拽排序 |
| 数据分析 | Excel/CSV 上传 + 统计图表 + 精细画像生成 |
| 可视化中心 | 雷达图 + 饼图 + 漏斗图 + 词云图 + 折线图 |
| 报告生成 | Word 报告 + 章节选择 + 图表嵌入 |

## 课程关联

本工具设计参考了《虚拟现实与游戏综合训练》课程中的用户认知设计理论：

- **玩家类型理论**：巴特尔模型（成就型/探索型/社交型/杀手型）
- **皮亚杰认知发展**：感知运动 → 前运算 → 具体运算 → 形式运算
- **游戏体验要素**：预感、完成、幸灾乐祸、赠与礼物、幽默感等
- **用户画像方法论**：WHO/WHAT/WHY 三维分析方法

## 技术栈

- **echarts-wordcloud** - 词云图
- **SheetJS (xlsx)** - Excel 处理

## 快速开始

### 方法一：直接打开

1. 下载 `index.html` 文件
2. 在浏览器中直接打开即可使用

### 方法二：本地服务器

```bash
# 克隆项目
git clone <repository-url>
cd game-user-research-tool

# 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080
```

## 使用说明

### 竞品分析
1. 从案例库选择游戏，或手动添加游戏
2. 对各维度进行评分（1-5分）
3. 填写文字描述（玩法特点、优势、劣势、建议）
4. 点击"生成雷达图"查看可视化对比

### 用户画像
1. 选择模板快速开始，或新建画像
2. 完成玩家类型测试，自动判定类型
3. 调整访谈问题顺序
4. 保存到本地或导出 JSON

### 用户访谈
1. 选择访谈提纲模板
2. 记录访谈内容
3. 从记录中提取洞察
4. 导出访谈报告

### 问卷设计
1. 选择问卷模板
2. 添加/编辑题目
3. 设置题目类型和维度
4. 导出 Excel 格式

### 数据分析
1. 上传 Excel/CSV 格式的问卷数据
2. 查看各题目统计数据
3. 生成精细画像

### 可视化中心
- 查看所有图表的集中展示
- 刷新图表更新数据
- 导出图表图片

### 报告生成
1. 选择报告类型
2. 选择要包含的章节
3. 点击生成 Word 报告

## 行业模板

| 品类 | 预置游戏 | 分析维度侧重 |
|------|----------|--------------|
| 二次元/卡牌 | 原神、崩坏：星穹铁道、明日方舟 | 角色魅力、抽卡体验、剧情深度、美术品质 |
| MOBA/竞技 | 王者荣耀、DOTA2、英雄联盟 | 公平竞技、段位体验、操作深度、社交组队 |
| 休闲/派对 | 蛋仔派对、糖豆人、元气骑士 | 上手门槛、碎片时间、社交乐趣、UGC 创作 |
| SLG/策略 | 万国觉醒、三国志战略版、率土之滨 | 资源平衡、联盟系统、付费深度、赛季节奏 |
| RPG/MMO | 剑网3、FF14、天涯明月刀 | 剧情沉浸、装备系统、社交生态、付费深度 |
| 射击/动作 | CSGO、守望先锋、Apex英雄 | 操作手感、皮肤系统、赛季模式、竞技平衡 |
